# Тестовое задание

Сборка выполнена на ubuntu с использованием библиотек libpqxx и libpq, БД - PostgreSQL

## Подключение к базе данных

1. Создайте файл **`db_config.ini`** в директории с исполняемым файлом.

2. Заполните его по следующему примеру со своими данными, соответствующие вашй стовк PostgreSQL:

   ```ini
   [database]
   name = postgres        ; имя базы данных
   user = postgres         ; имя пользователя PostgreSQL
   password = admin        ; пароль пользователя
   host = 127.0.0.1        ; адрес сервера БД
   port = 5432             ; порт подключения
